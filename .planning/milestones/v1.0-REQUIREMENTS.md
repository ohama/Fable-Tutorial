## Milestone Summary

**Shipped:** 20 of 20 v1 requirements ✅

**Adjusted:**
- **UI-03** — 원래 "Feliz.Router로 SPA 라우팅"이었으나 Feliz.Router 4.0.0이 핀된 Feliz 3.3.3과 비호환(Feliz 4.x 내부 API 의존)으로 확인됨. 동일한 관찰 가능 목표(URL 변경 시 뷰 전환)를 수제 hash 라우팅(window.location.hash + hashchange 구독)으로 달성. 챕터는 그 이유를 정직하게 서술.
- **SETUP-02** — "Option None → null" 가정이 실측 결과 `undefined`로 확정되어 ROADMAP SC-2와 본문을 정정.
- **REAL-01** — `Async.AwaitPromise`가 대규모 Elmish+Feliz+Thoth 스택에서 타입 체크 실패하여 의미가 동일한 `Cmd.OfPromise.either` + `Promise.bind`로 구현.

**Dropped:** 없음.

**Deferred to v2:** INFRA-05 (예제 CI 격리 빌드).
