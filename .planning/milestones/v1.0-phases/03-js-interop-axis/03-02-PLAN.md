Erased union facts (RESEARCH Pattern 3 + Pitfall 1 + Code Examples):
- U2-U9 are erased at runtime вҖ” only the raw JS value remains. Construct with `!^value` (needs `open Fable.Core.JsInterop`) or `U2.Case1 x` / `U2.Case2 x`.
- Pattern matching compiles to runtime type checks: `typeof` / `Array.isArray` / `instanceof`.
- SAFE `U2<string,int>`: Case1 вҶ’ `typeof x === "string"`, Case2 вҶ’ number. Distinct вҶ’ both branches reachable.
- UNSAFE `U2<int,float>`: BOTH int and float are JS `number`, so BOTH cases compile to `typeof x === "number"`. The FIRST branch always wins; the second is DEAD CODE. No Fable compiler error вҶ’ silent runtime bug.
- Safe-pair rules table (RESEARCH Pattern 3): string|int SAFE; string|float SAFE; int[]|string SAFE (Array.isArray); obj|string SAFE; int|float UNSAFE; int|uint32 UNSAFE; int|int64 SAFE (int64 is BigInt вҶ’ typeof "bigint"); string|bool SAFE.
- StringEnum: `[<StringEnum(CaseRules.KebabCase)>] type CssBoxSizing = ContentBox | BorderBox` вҶ’ ContentBox compiles to the bare string literal `"content-box"`, no runtime object. CaseRules: None, LowerFirst (default), SnakeCase, SnakeCaseAllCaps, KebabCase, LowerAll. `[<CompiledName("...")>]` overrides a single case.
- Note: actually CALLING handleUnsafe with a float at runtime would mis-match вҖ” but for a BUILD-only example, focus on the GENERATED typeof check being identical for both branches (visible by inspecting App.fs.js), which is the load-bearing demonstration.

// ANCHOR: safe-union
// SAFE: string vs int вҖ” typeof distinguishes them, so both branches are reachable.
let handleSafe (x: U2<string, int>) : string =
    match x with
    | U2.Case1 s -> sprintf "string: %s" s   // typeof x === "string"
    | U2.Case2 n -> sprintf "number: %d" n   // typeof x === "number"
// ANCHOR_END: safe-union

// ANCHOR: unsafe-union
// UNSAFE: int vs float вҖ” both are JS number, so BOTH cases compile to the same
// typeof x === "number" check. Case2 is DEAD CODE and never runs. No compiler error.
let handleUnsafe (x: U2<int, float>) : string =
    match x with
    | U2.Case1 n -> sprintf "int: %d" n      // typeof x === "number" вҶ’ ALWAYS matches
    | U2.Case2 f -> sprintf "float: %f" f    // unreachable
// ANCHOR_END: unsafe-union

// ANCHOR: stringenum
// StringEnum compiles to bare string literals вҖ” no runtime object.
[<StringEnum(CaseRules.KebabCase)>]
type CssBoxSizing =
    | ContentBox   // вҶ’ "content-box"
    | BorderBox    // вҶ’ "border-box"

let sizing : CssBoxSizing = ContentBox
// ANCHOR_END: stringenum

let app = document.getElementById "app"
app.innerHTML <-
    sprintf "<h1>кі кёү Interop</h1><p>%s</p><p>%s</p><p>box-sizing: %A</p>"
        (handleSafe !^"hello") (handleUnsafe !^7) sizing
